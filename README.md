# 🐻🍳 꼬미의 부엌

냉장고·식탁 재고를 기반으로 곰돌이 **꼬미**가 오늘 뭐 해먹을지 추천해주는 모바일 웹앱이에요.
스타듀밸리 같은 코지 도트 감성으로 만들었어요.

**👉 바로 열기:** https://jin2hyun-web.github.io/komi-kitchen/

## 화면
- 🏠 **홈** — 꼬미 인사 + 오늘의 추천 (곧 상하는 재료를 쓰는 메뉴를 우선 추천)
- 🥕 **내 부엌** — 냉장고·냉동고·팬트리·양념장 재고 관리, 신선도 표시
- ➕ **재료 추가** — 도트 재료 고르기 / 직접 입력 / 수량·위치·유통기한
- 🐻 **꼬미 상담** — 냉장고 재고를 다 아는 AI 꼬미와 대화 (Claude API)
- 👤 **MY** — API 키·모델 설정, 데이터 초기화

## AI 꼬미 켜는 2가지 방법

### A) CLI 두뇌 — 내 맥 + 폰 (API 키·결제 불필요) 🐻
내 Claude 구독으로 도는 로컬 서버. **앱이 로컬 서버면 자동으로 CLI 두뇌**를 써요.
```bash
python3 server.py            # 또는 '꼬미서버-켜기.command' 더블클릭
```
- 이 맥에서: `http://localhost:8787`
- **폰(같은 와이파이):** `http://<맥 LAN IP>:8787` (서버 켜질 때 주소를 알려줘요)
- 맥이 켜져 있고 서버가 떠 있어야 해요. 키 입력 없이 바로 대화돼요.

### B) API 키 — 어디서나 (GitHub Pages 공개 버전)
1. MY 화면 → **키 발급받기**에서 Anthropic API 키 발급 (`sk-ant-...`)
2. MY에 붙여넣고 **저장** → 꼬미 탭에서 대화 (기본 Haiku 4.5, 사용한 만큼 과금)

> 모든 데이터(재고·대화·키)는 **사용자 기기에만** 저장돼요. 서버로 전송되지 않아요.

## 파일 구성
| 파일 | 설명 |
|------|------|
| `index.html` | 앱 본체 (도트 에셋은 `assets/` 참조) |
| `assets/` | 도트 PNG 에셋 37개 |
| `꼬미의-부엌.html` | 에셋까지 전부 합친 **단일 파일** (폴더 없이 어디서든 열림) |
| `build_single.py` | 단일 파일 생성 스크립트 (`python3 build_single.py`) |
